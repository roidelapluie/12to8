# 12to8

A client for [9to5r](https://github.com/kalmanolah/925r)

# Install

```
$ go get github.com/Inuits/12to8
```

# Usage

List users:

```
12to8 list users
```

List timesheets:

```
12to8 list timesheets
```

Create new timesheet:

```
12to8 new timesheet
12to8 new timesheet 9
12to8 new timesheet 9/2017
```
